# Portfolio Pro - Elmas Oulobo

## Description
Professional portfolio website for a Data Science Master's student showcasing expertise in data analysis, automation, and AI.

## Features
- ✅ Bilingual support (French/English)
- ✅ Modern, responsive design
- ✅ 6 complete pages (Home, About, Services, Projects, Training, Contact)
- ✅ Framer Motion animations
- ✅ SEO optimized
- ✅ Professional color scheme (blue, gray, black, white)

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Quick Start
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm start
```
